# VPS Agency Optimization Project

## 🚀 Project Vision
Make our VPS-based AI agency truly awesome by optimizing agent intelligence, token utilization, system performance, and overall operational excellence.

## 🎯 Core Objectives
1. **Agent Intelligence Enhancement** - Make agents smarter, more autonomous, and context-aware
2. **Token Optimization** - Maximize efficiency of API token usage across all services
3. **System Performance** - Optimize VPS resources, response times, and reliability
4. **Operational Excellence** - Streamline workflows, monitoring, and maintenance
5. **Security & Scalability** - Enhance security posture and prepare for scaling

## 🔧 Git Setup (No More Amnesia)
To prevent agents from forgetting Git configuration between sessions:
```bash
# Run once per session
/root/jira-agency/scripts/git-setup.sh

# Or manually:
git config --global user.email "pm@jira-agency.ai"
git config --global user.name "Jira Agency PM"
git remote set-url origin git@github.com:aiagencycx-dot/vps-agency-optimization.git
```

**SSH Key**: `~/.ssh/github_aiagencycx` (pre-configured)
**Memory File**: `/root/jira-agency/memory/git-setup.md`
